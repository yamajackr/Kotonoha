AutoDefine - Automatically define vocabulary words with pronunciations and image

AutoDefine simplifies finding definitions and pronunciations for words. Just install AutoDefine, click the blue globe (or use the shortcut Ctrl+Alt+E), and see all pronunciations and definitions appear within the card instantly! Configuration options allow you to specify which field the pronunciation / definitions will be inserted in.

AutoDefine is optimized for use with an individual word/note at a time. If you want to generate many definitions at once, I recommend <a href="http://www.easydefine.com/">EasyDefine</a> (it scales well to hundreds of words).

Created by <a href="http://www.robertsanek.com">Robert Sanek</a>. Licensed under GPLv2. See <a href="https://github.com/z1lc/AutoDefine">GitHub</a> for the source. If you find any bugs, please <a href="https://github.com/z1lc/AutoDefine/issues">file an issue</a>.

<b>NOTE:</b> This add-on uses Merriam-Webster's Collegiate&#174; Dictionary with Audio API to get definitions and pronunciations. This requires you sign up for a Merriam-Webster account and use your own individual API access key. Go to the <a href="http://www.dictionaryapi.com/">Merriam-Webster Dictionary API website</a>, sign up for an account, and request access to the Collegiate Dictionary. Then, in Anki, go to Tools>Add-Ons>AutoDefine>Edit... and replace <code>YOUR_KEY_HERE</code> with your unique API key.
